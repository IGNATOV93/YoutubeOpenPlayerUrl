Программа консольного типа для запуска видеороликов с ютюб на вашем пк(видеоплеер) через вашего телеграм бота.
Я использовал плеер "PotPlayer" ,настроив заранее во весь экран при открытии видео.Можете использовать другой видеоплеер с поддежкой url.
Для работы создайте ini файл в папке где запускается программа с именем "tokens.ini" ,PatchOftePlayr:здесь пишите полный путь для запуска вашего плеера,
NameProcessPlayer:здесь пишите название процесса вашего плеера.
Структура Ini файла внутри :


[Profile0]

YourBotTelegreamToken=здесь пишите токен вашего бота телеграм

PatchOftePlayr=C:\\Program Files\\PotPlayer\\PotPlayerMini64

NameProcessPlayer=PotPlayerMini64
